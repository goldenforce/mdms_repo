Introduction

Mapping Designer is a tool used for creating connections. It is based on eclipse framework and plugging into the GoldenSource Workbench. It supports the MDX (Mapping Definition Extended) format. It also has the feature to convert the old MSF connections into MDX.

Installation Instructions

For a detailed installation description refer to chapter "Install GoldenSource Workbench and PlugIns" of the "GoldenSource Enterprise Installation Guide".
